# Smart Password Generator

## Description

The Smart Password Generator is a Java application that generates secure passwords using user details such as Name, PAN Number, and Date of Birth. The generated password is then analyzed to determine its strength.

## Features

* Generate password automatically
* Uses Name, PAN Number, and DOB
* Password strength analysis
* Strong, Medium, and Weak classification

## Technologies Used

* Java
* String Manipulation
* Regular Expressions

## Sample Input

Name: Tushar Singh

PAN Number: ABCDE1234F

Date of Birth: 15/08/2004

## Sample Output

Generated Password: TUS1234@#2004

Password Strength: Strong

## Concepts Used

* String Handling
* Arrays
* Conditional Statements
* Pattern Matching
