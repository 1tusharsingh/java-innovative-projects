# Text Steganography System

## Description

Text Steganography System is a security-based Java application that hides a secret message inside a text file and retrieves it when required. The project demonstrates file handling and information hiding techniques.

## Features

* Accept cover file and secret message
* Hide message inside text file
* Save encoded file
* Retrieve hidden message
* Support multiple test cases

## Technologies Used

* Java
* File Handling
* String Processing
* BufferedReader
* BufferedWriter

## Sample Input

Cover File: cover.txt

Output File: encoded.txt

Secret Message: Meeting at 10 AM

## Sample Output

Message Hidden Successfully!

Retrieved Message: Meeting at 10 AM

## Concepts Used

* File Handling
* String Manipulation
* Exception Handling
* Data Hiding Techniques
