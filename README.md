# Projects Included

## 1. Smart Password Generator

### Description

The Smart Password Generator is a security-focused Java application that generates strong passwords using user details such as name, PAN number, and date of birth. The application also evaluates the strength of the generated password and categorizes it as Strong, Medium, or Weak.

### Features

* Accepts user details as input
* Generates passwords automatically
* Password strength evaluation
* Uses string manipulation techniques
* Implements pattern matching using regular expressions

### Concepts Used

* String Handling
* Arrays
* Regular Expressions
* Conditional Statements
* User Input Handling

### Sample Output

```text
===== Smart Password Generator =====

Enter Name: Tushar Singh
Enter PAN Number: ABCDE1234F
Enter Date of Birth (DD/MM/YYYY): 15/08/2004

Generated Password: TUS1234@#2004
Password Strength: Strong
```

## 2. Personal Expense Tracker

### Description

The Personal Expense Tracker is a financial management application that helps users record and monitor daily expenses. The system allows users to store expense information, generate reports, identify spending patterns, and save data for future use.

### Features

* Add daily expenses
* Display all recorded expenses
* Generate monthly expense reports
* Identify highest expense category
* Save expense records to a file
* Load previous expense records automatically

### Concepts Used

* ArrayList
* HashMap
* File Handling
* Object Serialization
* Menu-Driven Programming
* OOP Concepts

### Sample Output

```text
===== PERSONAL EXPENSE TRACKER =====

1. Add Expense
2. Display Expenses
3. Monthly Expense Report
4. Highest Expense Category
5. Save Data
6. Exit
```

### Learning Outcomes

* Managing collections of objects
* Working with files and persistent storage
* Report generation techniques
* Data analysis using Java collections

## 3. Text Steganography System

### Description

The Text Steganography System is a security-based application designed to hide and retrieve secret messages within a text file. The project demonstrates how information can be concealed and later extracted using Java file handling and string processing techniques.

### Features

* Accept cover file and secret message
* Hide messages inside a text file
* Save encoded output file
* Retrieve hidden messages
* File-based message processing
* Multiple test case support

### Concepts Used

* File Handling
* String Processing
* BufferedReader
* BufferedWriter
* Exception Handling

### Sample Output

```text
Enter Cover File Name: cover.txt
Enter Output File Name: encoded.txt
Enter Secret Message: Meeting at 10 AM

Message Hidden Successfully!

Retrieving Hidden Message...

Hidden Message: Meeting at 10 AM
```

### Learning Outcomes

* Secure data handling concepts
* File processing techniques
* Text manipulation
* Practical implementation of information hiding
