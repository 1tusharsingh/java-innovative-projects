# Personal Expense Tracker

## Description

Personal Expense Tracker is a Java-based application that helps users manage daily expenses. The system stores expense records, generates reports, identifies spending patterns, and saves data for future use.

## Features

* Add expense details
* Display all expenses
* Generate monthly expense reports
* Find highest expense category
* Save data using file handling
* Load previous records automatically

## Technologies Used

* Java
* ArrayList
* HashMap
* File Handling
* Object Serialization
